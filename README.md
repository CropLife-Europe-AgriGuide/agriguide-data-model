# AgriGuide Data Model

The AgriGuide Data Model is the open data standard underpinning **AgriGuide**, CropLife Europe's digital labelling platform for plant protection products (PPPs) across the European Union. It provides a common, machine-readable structure for representing crop protection product label information in a consistent form.
This repository publishes the model so that regulators, member companies, farm management information system (FMIS) providers, standards bodies and other interested parties can reference, adopt and align with a single, canonical representation.

## Why this is published

Publication is driven by three objectives:

* **Interoperability** with farm management information systems, the OECD Communities of Practice, and relevant ISO standards work.
* **Ecosystem positioning** — to signal CropLife Europe's stewardship of the standard and to invite collaboration from regulators, member companies and adjacent providers.
* **Regulatory readiness** ahead of the 2028 and 2030 mandatory digital label deadlines under EU Regulation 2023/564.

CropLife Europe is not commercialising the data model itself. The model is open, enabling wider collaboration with interested parties across Europe and globally.

## What this repository contains

The initial release (v 1.2) comprises:

* JSON-LD schema files
* Ontology files
* Documentation
* The AgriGuide overview deck and introductory slides on the model
